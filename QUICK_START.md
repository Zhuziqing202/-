**或者用简单的方式：**


### 4️⃣ 如何添加项目图片

**简单方式：**
1. 创建 `images/` 文件夹在网页目录
2. 将图片放进去（如 `project1.jpg`）
3. 在页面中找到：
```html
<div class="image-placeholder">
  [项目相关的图片、图表或可视化内容]
</div>
```

4. 替换为：
```html
<div style="margin: 2rem 0;">
  <img src="../images/project1.jpg" alt="项目名称" style="width: 100%; border-radius: 12px;">
</div>
```

### 7️⃣ 常见问题快速解答

**Q: 我改了内容但网站没有更新？**
A: 按 `Ctrl+Shift+R` 强制刷新浏览器缓存

**Q: 链接无法点击？**
A: 检查 `href="..."` 的路径是否正确。如果在 `projects/` 里，到 `pages/` 的链接应该是 `href="../pages/thoughts.html"`

**Q: 手机上显示不全？**
A: 这说明有CSS问题。检查是否修改了 `<style>` 部分

**Q: 如何修改网站的颜色？**
A: 在任何HTML文件的 `:root` 中改这行：
```css
--color-accent: #4f46e5;  /* 改成你想要的颜色 */
```

推荐颜色：
- `#0ea5e9` - 蓝色
- `#10b981` - 绿色  
- `#f59e0b` - 琥珀
- `#ec4899` - 粉色

**Q: 如何删除某个项目？**
A: 在 `index.html` 的 projects-grid 中删除对应的 `<a class="project-card">...</a>` 块

**Q: 我想添加第5个项目？**
A: 1. 复制 `projects/tech-2.html` 为 `projects/tech-3.html`
   2. 修改内容
   3. 在 `index.html` 的项目网格中添加新卡片链接
